# javaweb-图书管理系统
技术栈:
Mybatis Thymeleaf Lombok Servlet Html+CSS+JavaScript
使用了Tomcat10作为本地服务器 Maven配置项目  MySql数据库    
实现功能：
- 图书管理员的登陆和退出（只有登陆之后才能进入管理页面）
- 图书的列表浏览（包括书籍是否被借出的状态也要进行显示）以及图书的添加和删除
- 学生的列表浏览
- 查看所有的借阅列表，添加借阅信息
